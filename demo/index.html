Orchard Splicer 🍎 ➜ 🍊 - Convert Between Ripchord & Scaler Presets

  • Back to Preset World

Convert between Plugin Boutique's Scaler 2 and Trackbout's Ripchord chord pack formats.

Scaler lacks names for chords, so tonal is used to determine chord names, and falls back to the keys used.

Scaler also lacks a trigger, so the trigger is set to auto incriment starting from C3 (48).

To convert files, choose them using the selector below and then click the Download link from the genereated output.
